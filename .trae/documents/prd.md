## 1. 产品概述

组件库自动文档站点生成工具——基于 Next.js 14 + TypeScript + MDX 构建，从组件源码自动提取 Props 定义，生成交互式文档站点。面向组件库开发者和使用者，解决文档编写效率低、与源码不同步、缺少交互预览的问题。

## 2. 核心功能

### 2.1 功能模块

1. **组件文档页**：组件标题与描述、交互演示区、代码示例、Props API 表格、变更日志
2. **全局导航**：左侧导航树按组件分类组织
3. **搜索功能**：全文搜索组件和文档内容
4. **主题系统**：暗黑/亮色主题切换 + 自定义主题配置（主色调、字体、圆角）
5. **Props 自动提取**：从 TypeScript 源码使用 react-docgen-typescript 解析接口

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 文档首页 | Hero 区域 | 展示组件库介绍、快速开始指引 |
| 组件文档页 | 标题与描述 | 组件名称、版本、功能描述 |
| 组件文档页 | 交互演示区 | 实时修改 Props 值预览组件效果 |
| 组件文档页 | 代码示例 | 多个示例切换，代码高亮显示 |
| 组件文档页 | Props API 表格 | 属性名、类型、默认值、是否必填、描述 |
| 组件文档页 | 变更日志 | 组件版本历史和变更记录 |
| 主题配置页 | 主题编辑器 | 自定义主色调、字体、圆角 |
| 搜索 | 全文搜索 | 搜索组件名称和文档内容 |

## 3. 核心流程

用户访问文档站点 → 通过左侧导航或搜索找到组件 → 查看组件描述 → 在交互演示区调整 Props 实时预览 → 切换代码示例查看用法 → 查看 Props 表格了解 API → 查看变更日志了解版本变化

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：靛蓝（Indigo）系 + 琥珀色点缀
- 次要色：灰色系中性色
- 按钮风格：圆角适中，柔和阴影
- 字体：JetBrains Mono（代码）+ 系统无衬线字体（正文）
- 布局风格：左侧固定导航 + 右侧内容区，卡片式内容块
- 图标风格：线条图标（Lucide）

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 文档首页 | Hero 区域 | 渐变背景、大标题、特性卡片、快速开始按钮 |
| 组件文档页 | 标题区域 | 组件名 H1、描述文本、版本标签 |
| 组件文档页 | 交互演示区 | 白色卡片容器、Props 控制面板、实时预览区 |
| 组件文档页 | 代码示例 | 标签页切换、语法高亮代码块、复制按钮 |
| 组件文档页 | Props 表格 | 条纹表格、类型标签高亮、必填标记 |
| 组件文档页 | 变更日志 | 时间线布局、版本号标签 |
| 全局布局 | 侧边栏导航 | 树形结构、分类折叠、高亮当前项 |
| 全局布局 | 顶栏 | 搜索框、主题切换按钮 |
| 主题配置页 | 主题编辑器 | 颜色选择器、字体下拉、圆角滑块、实时预览 |

### 4.3 响应式设计

- 桌面优先设计，侧边栏常驻
- 平板端侧边栏可折叠
- 移动端侧边栏隐藏，通过汉堡菜单展开
